# FastApi

**1. Install python 3.6+**
**2. Setup virtual environment through venv then activate it**
```
python -m venv <environment-name>
source <environment-name>/bin/activate
```
**3. Install fastapi**
```
pip install fastapi
```
**4. Install uvicorn**
```
pip install uvicorn
```
**5. Start server**
```
uvicorn main:app --reload
```
